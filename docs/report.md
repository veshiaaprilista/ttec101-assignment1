# TTEC101 Assignment 1

## Project Title
Programming Foundations & Reproducible Workflow

## Dataset Description
The data sets required the age, gender, study hour,, attandance rate, assignment score, final score of 200 students.

## Objectives
- Learn how to use Git and GitHub
- Learn basic command line tools
- Learn how to organize project files
- Learn basic R for data analysis
- Understand reproducible workflow in data analytics

## Steps Taken
- Created project folder using terminal
- Created subfolders (data, docs, scripts)
- Placed dataset into data folder
- Created report and R script files
- Initialized Git repository
- Used git add and git commit to save changes
- Connected local repository to GitHub
- Pushed project to GitHub successfully

## Reflection
In this assignment, I learned how to use Git and GitHub to manage my project files. I also learned basic command line skills such as creating folders and files. This assignment helped me understand how data analytics projects are organized step by step.
